浏览器插件推荐

1. Nooboss（二管家）

一个管理浏览器插件的插件。

2. Tampermonkey（油猴）

管理脚本的插件。

3. uBlock Origin

广告拦截

4. History Manager

管理浏览历史，虽说浏览器自带有，我觉得这个比较方便，还有历史记录的统计数据。

5. Fatkun图片批量下载 - Pro

下载网页图片的，包括jpg、png、gif等格式，下载时将网页下拉到底，保证图片全部加载，这时点击插件就可以下载当前页面或所有页面，会弹出一个新页面，可以对图片进行排序，筛选，建议勾选重命名，后面无需改，这样下载下来图片就是按照顺序排的。

下载前的提示一定要注注注注注注注注注注意意意意意意意意意意

6. FireShot

网页截图。有的浏览器也会自带，功能也没什么特殊，就是不能截长图
有的浏览器ctrl + m可以截全网页，自己可以试试，或者借助其它软件。

9. Onetab

可以将浏览器打开的标签页暂时保存下来。

12. 浏览器书签搜索

Holmes

https://blackfish.fi/holmes/

就浏览器书签很多不好找，或者要的网站在多级文件夹下面点击很麻烦，直接搜索会快一点。

后来才知道浏览器地址栏也能搜索书签，我试一下，有时候输一个字的时候没有显示出来。

大家可以试一下，挑着用吧。

使用：直接点击插件，会弹出一个搜索框，输入关键词就行，不过最多只能出来十个结果。

13. 网络爬虫

Web scraper

https://www.webscraper.io/

如果要爬虫，一般都是用python，不过需要一段时间的学习，而这款插件不用编程，基本靠点击操作就能完成简单的数据爬取。

13.1. 使用方法：

以爬取豆瓣电影top250为例。

安装完插件，打开top250首页：

https://movie.douban.com/top250?start=0&filter=

按F12打开控制台，可以看到最后出现了web scraper

创建一个sitemap，相当于新建一个爬取任务。

输入任务名称和地址

观察一下地址

所以URL填入

进入任务，首先增加一个selector（选择器）

Type选择Element（元素），Multiple选项勾上（因为要爬取多条数据）

鼠标移到上面所选区域会变色，选取前两条后面全部就会选上

Done selecting后，可以预览一下，看下是否全部选上

保存

进入这个选择器，再增加一个选择器，

同理，也可以增加电影名、电影简介、电影评分等。

完成后，可以观察选择器的逻辑关系

然后开始爬取

开始后会弹出一个窗口，爬取结束后自动关闭。

刷新一下就能看到爬取的数据

可以导出为csv文件。

爬取结束
