使用hexo+github搭建个人博客

1. 安装Node.js

地址：http://nodejs.cn/download/

2. 安装Git

地址：https://git-scm.com/downloads

2.1. 验证安装是否成功

使用 Win+R cmd 打开命令行

会显示各个版本，如图：

3. 安装hexo

3.1. 验证

结果如图：

4. 初始化hexo

在想要存放博客文件的目录下，右键，选择git bash here

blog为创建的文件夹名

安装完成后会生成以下内容，主要为：

文件名 描述
node_modules 依赖包（插件）
scaffolds 生成文章的一些模版
source 你的文章目录
themes 主题
_config.yml 配置文件
public 静态页面目录，后面会生成

根据md文件生成静态页面，在生成的public的文件夹里

打开本地服务
浏览器打开http://localhost:4000即可看到生成页面，会比较慢

页面显示如下即成功：

Ctrl+C关闭服务

5. 创建github仓库

5.1. 注册github

地址：https://github.com

5.2. 创建仓库

Repository name填上

name.github.io

例如

sensen1369.github.io

5.3. 连接电脑和github帐号

打开git

之后一直回车确定，完成后会在c盘用户目录下生成.ssh文件夹

用记事本打开id_rsa.pub，全选复制
进入github主页，点击右边头像，点击settings
新建ssh key

title随便填，key粘贴刚才复制的内容

测试连接

如图即为成功

6. hexo部署

打开blog文件夹下的_config.yml文件，拉到最后，修改

deploy:
    type: git
    repo: https://github.com/name/name.github.io.git
    branch: main

name是创建的仓库名字

6.1. 安装deploy-git

是部署命令，可以将文件上传到github仓库

清除之前生成的页面，即删除public文件夹

生成页面，部署页面

7. 完成

浏览器打开

https://name.git.io

即可看到页面

8. hexo配置文件修改

博客根目录下的_config.yml是整个hexo框架的配置文件，详细可参考官方的配置描述。

8.1. Site

名称 描述
title 网站标题
subtitle 网站副标题
description 网站描述
keywords 关键词
author 作者
language 网站语言
timezone 时区

修改建议：

title: 
author: 
language: zh_CN

注意：
配置’ : ‘后要加空格

8.2. URL

名称 描述
url 网站
permalink 网站链接格式

修改建议：

permalink:
参数 结果
:year/:month/:day/:title/ 2020/06/10/hello-world/
:year/:month-:day-:title/ 2020/06-10-hello-world/

8.3. Extensions

下面就有个themes，配置页面主题，默认landscape主题。

9. 更改主题

官网主题地址：https://hexo.io/themes/
或者去github搜索hexo theme都能找到一大推主题

以hexo-theme-fluid为例
项目地址为：https://github.com/fluid-dev/hexo-theme-fluid

项目下方都会写明使用方法

以下简单介绍一下：

进入网站根目录blog，右键git bash here

修改博客配置文件_config.yml，注意是根目录下的，不是主题目录下的

themes: hexo-theme-fluid

本地检查是否有问题，没问题的话上传

10. 创建新博客

会根据根目录下的scaffolds内文件生成md文件，默认按照post.md

可在配置文件_config.yml修改

10.1. 编写markdown

推荐使用VScode

下载地址：https://code.visualstudio.com/

10.2. 浏览器预览

预览地址:https://localhost:4000

10.3. 生成静态页面

10.4. 提交到github

11. 博客插入图片

11.1. 安装插件

11.2. 打开博客配置文件_config.yml

Writing栏下，修改

post_asset_folder: True

11.3. 新建博客

在source/_posts目录下会生成一个pic.md和pic文件夹

将图片放入pic文件夹内

打开pic.md文件，编写说明：

![](./pic/sensen.png)

注意使用相对路径

12. 文章标题自动编号

项目地址：https://github.com/legendmohe/markdown_index

vscode搜索插件markdown-index安装

命令面板输入

markdown add index

运行即可添加编号

重复运行即可更新编号
